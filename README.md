# Mediatech Project

This is a full-stack application for managing clients, products, and invoices.

## Structure

The project is divided into two main parts:

-   **mediatech** (Backend): A Spring Boot application providing the REST API.
-   **mediatech-ui** (Frontend): An Angular application for the user interface.

## Prerequisites

-   Java JDK 17+
-   Node.js & npm
-   Maven

## How to Run

### Backend
1.  Navigate to the `mediatech` directory.
2.  Run `mvn spring-boot:run`.

### Frontend
1.  Navigate to the `mediatech-ui` directory.
2.  Run `npm install` (first time only).
3.  Run `npm start` or `ng serve`.
4.  Access the app at `http://localhost:4200`.
